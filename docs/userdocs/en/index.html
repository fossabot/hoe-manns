Abstract

hoe-manns extends the standard Rake-Task with some useful new tasks.

⁠1. Introduction to hoe-manns

hoe-manns is a extension of the Ruby projectsoftware hoe (see https://github.com/seattlerb/hoe.) It extends the standard Ruby tasks to some new useful tasks.
To use the software you will need a installed Ruby version (Higher or equal 2.2.0), parseconfig, bundler-audit and a installed and already configured hoe.

⁠5. Additional Information

The API-Dokumentation is placed there: http://www.rubydoc.info/gems/hoe-manns.
You can place Bugreports there: https://bugs.launchpad.net/hoe-manns.
You can place Feature Requests there: https://blueprints.launchpad.net/hoe-manns.

Caution

This software was developed and tested on a Linux system with RVM as Rubymanager. If anyone has any idea to make the software fit for other OS and Rubymanagers, then you can send a Pull-Request on Github.
